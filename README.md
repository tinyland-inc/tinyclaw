# tinyclaw

This repository is maintained as a downstream Tinyland collaboration fork of
[`Jesssullivan/tinyclaw`](https://github.com/Jesssullivan/tinyclaw).

Canonical development, issue intake, releases, and current status tracking
happen at the Jess-owned repo:

- [`Jesssullivan/tinyclaw`](https://github.com/Jesssullivan/tinyclaw)

## Routing

- Open new issues in the canonical repo.
- Open pull requests against the canonical repo for active development.
- Do not open the same issue on both the canonical repo and this fork.

## What This Fork Is For

- downstream Tinyland visibility
- org-side collaboration when explicitly needed
- historical linkage to the broader Tinyland agent ecosystem

This fork may lag the canonical repo and should not be treated as the primary
source of truth.
